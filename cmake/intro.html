Overview¶

Overview¶

BDE build system is a set of tools and configuration files that uses cmake generator to produce make files for the low level build system (like ninja or make) and invoke this low level system to build, install and test the BDE libraries.

The workflow consists of the following steps:

Configuration
cmake parses the build description files, resolves internal and external dependancies and produces build, test and install targets
Build
cmake invokes the low-level build system to compile and link the BDE libraries and unit tests.
Test
Runs unit tests and present the results.
Install
Installs the build artifacts (header files, libraries, applications and supporting metadata) to the specified location.
